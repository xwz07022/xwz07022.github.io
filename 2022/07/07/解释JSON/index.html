JSON理解

JSON VS XML

JSON是一种轻量级的数据交互格式。XML是扩展标记语言，XML是一种像HTML语言。

JSON和XML的不同点:XML需要使用XML解析器来解析，JSON可以使用标准的JavaScript函数来解析。

XML比JSON更难解析；JSON可以直接使用现有的JavaScript对象解析；针对AJAX应用，JSON比XML数据加载更快，而且更简单；
